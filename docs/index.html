Features

⚡

In-Memory Caching

GET responses are cached automatically with configurable TTL. Repeat requests are served instantly.

📦

Zero Dependencies

Built entirely with Node.js built-in modules. No npm install needed beyond the project itself.

🔍

Cache Visibility

Every response includes an X-Prozy-Cache header so you always know if it was a cache hit or miss.

📊

Built-in Stats

Hit /stats to see all cached entries and their ages. Hit /health for a quick status check.

🔐

HTTPS Support

Transparently proxies both HTTP and HTTPS targets with automatic protocol detection.

⚙️

Configurable

Customize port, cache TTL, and max cache size through simple environment variables.

Quick Start

1

Clone the repo

git clone https://github.com/your-repo/prozy.git
2

Start the server

npm start
3

Send requests through the proxy

curl http://localhost:8080/http://example.com

Configuration

Variable Default Description
PORT 8080 Server listening port
CACHE_TTL 300 Cache lifetime in seconds
MAX_CACHE_SIZE 100 Maximum number of cached responses
